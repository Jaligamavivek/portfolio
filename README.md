# Jaligama Vivek - Portfolio Website

A modern, responsive portfolio website showcasing skills, projects, and experience.

## Setup Instructions

### Adding Your Profile Image

1. Save your profile image as `profile.jpg` in the `images/` folder
2. The image will automatically appear in the circular frame on the hero section
3. Recommended image size: 300x300 pixels or larger (square format works best)

- Modify colors in the CSS file to match your brand
